# ASTRANUMERICA — GERAÇÃO DE PDFs PERSONALIZADOS

## COMO FUNCIONA
Os PDFs são gerados automaticamente com base
nas respostas do quiz do funil.

## VARIÁVEIS DO QUIZ
- {nome}
- {signo}
- {sexo} → masculino / feminino
- {dor} → financas / amor / saude
- {estado_civil}
- {faixa_etaria}

## ARQUIVOS QUE MUDAM POR LEAD
- Capa com nome e signo
- Bloco de abertura (sexo)
- Bloco da dor (financas / amor / saude)
- Cases (financas / amor / saude)
- Visualização futura (financas / amor / saude)

## ARQUIVOS QUE SÃO FIXOS
- As 3 etapas do ritual
- Mapa hermético 78 páginas
- Os 3 bônus
- Preço e garantias
- Urgência e escassez

## ONDE FICAM OS OUTPUTS
astranumerica-pdfs/outputs/

## NOMENCLATURA DOS ARQUIVOS GERADOS
produto-principal_{sexo}_{dor}_{nome}.pdf
