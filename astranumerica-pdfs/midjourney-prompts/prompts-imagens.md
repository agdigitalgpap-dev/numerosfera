## NOTAS TÉCNICAS

- Usar sempre fundo escuro (deep blue, indigo, midnight, dark navy) para consistência com a paleta da marca
- Paleta dominante: dourado (#FFD700), roxo profundo (#4B0082), azul meia-noite (#191970)
- Após gerar, fazer upscale (U1-U4) para maior resolução
- Variações recomendadas: usar /vary para 2-3 variações de cada imagem importante e escolher a melhor
- Para elementos sem fundo: usar Photoshop/Canva para remover fundo após geração
