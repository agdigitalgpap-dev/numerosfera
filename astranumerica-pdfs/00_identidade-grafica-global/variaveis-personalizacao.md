# VARIÁVEIS DE PERSONALIZAÇÃO — QUIZ

Todos os PDFs gerados devem ser personalizados
com base nas respostas do quiz. As variáveis abaixo
devem ser injetadas em cada arquivo gerado.

## VARIÁVEIS DISPONÍVEIS
- {nome} → Nome da pessoa
- {signo} → Signo astrológico
- {sexo} → Masculino / Feminino
- {dor} → Finanças / Amor / Saúde
- {estado_civil} → Solteiro(a) / Casado(a) / Divorciado(a) / Viúvo(a)
- {faixa_etaria} → Jovem / Adulto / Maduro

## COMBINAÇÕES POSSÍVEIS POR PDF
Cada PDF deve ter versões para:

SEXO (2 versões):
- Masculino
- Feminino

DOR (3 versões):
- Finanças
- Amor
- Saúde

TOTAL DE COMBINAÇÕES: 6 versões por PDF
